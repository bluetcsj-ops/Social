# 世界杯 2026 社媒推广助手使用说明

这是一个“半自动”的社媒内容系统，用来生成文案草稿和分享图，方便你手动审核后发布到 X、Facebook、Discord 或 Telegram。

它不会登录你的账号，不会自动发帖，也不会循环刷屏。

## 主要文件

- `social-dashboard.html`：本地中文操作界面。
- `social/generate_social.py`：重新生成草稿和分享图的脚本。
- `social/drafts/x-posts.json`：X 文案草稿。
- `social/drafts/facebook-posts.json`：Facebook 文案草稿。
- `social/drafts/discord-posts.json`：Discord / Telegram 文案草稿。
- `social/calendar/30-day-calendar.json`：30 天发布节奏建议。
- `social/images/`：球队预测分享图。

## 打开仪表盘

可以直接双击打开：

```text
J:\promotion helper\social-dashboard.html
```

页面会优先读取 `social/social-dashboard-data.js`，所以即使本地服务器没有启动，也能显示草稿。

也可以用本地服务器打开：

在项目目录 `J:\promotion helper` 里运行：

```powershell
python -m http.server 8765 --bind 127.0.0.1
```

然后在浏览器打开：

```text
http://127.0.0.1:8765/social-dashboard.html
```

如果 8765 端口被占用，可以换成其他端口，例如：

```powershell
python -m http.server 8899 --bind 127.0.0.1
```

对应打开：

```text
http://127.0.0.1:8899/social-dashboard.html
```

如果浏览器跳到其他页面，或者提示打不开 `127.0.0.1`，直接重新打开 `social-dashboard.html` 文件即可。

## 基本使用流程

1. 打开 `social-dashboard.html`。
2. 用左侧筛选器选择平台、分类或状态。
3. 查看文案和图片预览。
4. 需要的话，直接在文本框里手动微调文案。
5. 点击“复制文案”。
6. 如果草稿带分享图，点击“复制图片”或“下载图片”。
7. 点击“打开 X”或“打开 Facebook”，进入平台的发布或分享页面。
8. 手动粘贴文案，并粘贴或上传图片，再发布。
9. 发布后回到仪表盘，点击“标记已用”。

“已使用”状态保存在当前浏览器的 `localStorage` 里，只影响你的本机浏览器。

## 重新生成草稿和图片

如果你修改了生成脚本，或者想恢复默认草稿，可以运行：

```powershell
python social\generate_social.py
```

脚本会重新生成：

- 30 条 X 草稿
- 20 条 Facebook 草稿
- 28 条 Discord / Telegram 草稿
- 10 个热门球队的 1200x630 分享图
- 10 个热门球队的 1080x1350 竖版图
- 30 天发布节奏建议
- 仪表盘离线数据文件 `social/social-dashboard-data.js`

注意：重新生成会覆盖 `social/drafts/`、`social/images/` 和 `social/calendar/` 中对应的输出文件。

## 发布节奏建议

可以参考 `social/calendar/30-day-calendar.json`：

- 每天最多 1 条 X。
- 每周 2 到 3 条足球讨论。
- 每周 1 条轻量网站相关内容。
- 每周 1 条球队预测或分享卡。

建议发布前都手动改几处，让语气更像你本人。

## 安全规则

生成脚本内置了基础反垃圾规则：

- 不生成完全相同的文案。
- X 每条最多 2 个 hashtag。
- 不使用“Please share”“Please upvote”等求赞求转发措辞。
- 不使用夸张销售话术。
- 大部分内容围绕足球讨论，而不是一直推广网站。

## 图片使用

横版图适合 X / Facebook：

```text
social/images/brazil-prediction.png
```

竖版图适合 Story、Telegram 或移动端分享：

```text
social/images/brazil-prediction-story.png
```

发布时可以先复制文案，再手动上传对应球队的图片。

### 为什么图片不能自动附上？

X 和 Facebook 的普通分享链接只能预填文字或链接，不能把本地图片文件自动塞进发帖框。要自动上传图片通常需要登录账号、调用平台 API 或控制浏览器操作，这会接近自动发帖机器人，不符合这个工具的安全目标。

所以仪表盘提供了更安全的手动流程：

- “复制图片”：把图片复制到剪贴板，之后在发帖框里粘贴。
- “下载图片”：保存或打开图片文件，再手动上传。
- “复制图片路径”：方便你在文件选择器里快速定位对应图片。

如果浏览器不支持直接复制图片，按钮会自动退回为复制图片路径。

## 推荐工作方式

先从“投票话题”和“球队预测”开始发，这类内容更像自然讨论。网站相关文案建议一周一次左右，保持轻量、真实、球迷口吻。
