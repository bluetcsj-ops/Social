# 真实球星图片素材

这里用于放置已经确认可使用的真实球星照片或合成图。

规则：

- 不使用 AI 生成的虚构球员脸作为球星图。
- 真实球星必须来自可使用的照片来源，例如你自己购买/授权的素材、官方允许使用的媒体包、或明确许可的开放授权图片。
- 保存文件时建议使用清晰命名：

```text
vini-brazil-hero.jpg
messi-argentina-hero.jpg
mbappe-france-hero.jpg
son-japan-hero.jpg
```

建议每张图保留来源记录：

```text
文件名：
球星：
来源：
许可：
下载日期：
备注：
```

如果暂时没有真实授权图，内容助手应该使用奖杯、球场、国旗、球迷氛围图片，而不是生成一个不存在的球员。
